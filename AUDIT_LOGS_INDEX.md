## 📋 Termes Clés

### Interfaces
- **Page Web**: `/admin/audit-logs/`
- **Admin Django**: `/admin/core/auditlog/`
- **API REST**: `/api/audit-logs/`

### Modèle
- **AuditLog**: Table contenant les logs d'audit
- **Champs**: table_name, operation, record_id, user_id, action_time, old_data, new_data, previous_hash, current_hash

### Opérations
- **INSERT/CREATE**: Création d'enregistrement
- **UPDATE**: Modification d'enregistrement
- **DELETE**: Suppression d'enregistrement
- **SELECT**: Lecture (non loggée)

### Permissions
- **@login_required**: Page web
- **IsAdminUser**: API
- **staff_member_required**: Admin Django

### Concepts
- **Filtrage**: Recherche par critères
- **Pagination**: Affichage par pages (50/page)
- **Hash**: Vérification d'intégrité SHA256
- **JSON**: Format de données pour old_data/new_data
